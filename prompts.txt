**Prompt 16 — Railway deployment**

```
Finalize Railway deployment:

1. railway.toml:
   Build: pip install -r requirements.txt
   Start: uvicorn main:app --host 0.0.0.0 --port $PORT
   Health check: GET /health

2. On first boot, download FinBERT model and cache
   to Railway volume. Never re-download on redeploy.

3. .env.example:
   SUPABASE_URL, SUPABASE_KEY,
   FINNHUB_API_KEY, TRADIER_API_KEY,
   TELEGRAM_BOT_TOKEN, TELEGRAM_CHAT_ID,
   TZ=America/New_York

4. Startup validation — check all env vars present
   and all APIs reachable before first run. Log
   clearly which key is missing and exit gracefully.

5. Add a /health response that shows connectivity
   status for every external API so you can debug
   from Railway logs instantly.
```
